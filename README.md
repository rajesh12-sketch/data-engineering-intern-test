# Data Engineering Internship Test

This project analyzes referral program data and validates referral rewards using business logic.

## Tools Used
- Python
- Pandas

## Steps to Run

1. Install Python
2. Install pandas

pip install pandas

3. Run the script

python analysis.py

The script will generate the final_report.csv file.

## Output
final_report.csv contains the processed referral data and business logic validation results.
